💳 ATM-Simulation
📌 Description

This is a simple ATM Simulation project in Python.
It mimics the basic functions of a real ATM machine like checking balance, depositing money, withdrawing money, and viewing transaction history.

This project is created for learning purposes and does not involve real banking operations.

🚀 Features
Display account balance
Deposit money
Withdraw money
View transaction statement
Menu-driven program

🛠️ Technologies Used
Python (Core Programming)
File Handling (for storing transactions)
